# aimemory

`aimemory` 是一个基于 VitePress 的文档站项目，用于承载：

- 外部 artifacts 文档
- Hermes 记忆快照
- 项目化方案与部署说明

推荐域名：`memory.oldphoto.site`

## 本地开发

```bash
npm install
npm run docs:dev
```

## 构建

```bash
npm run docs:build
npm run docs:preview
```

## 目录

- `docs/`：站点内容
- `docs/.vitepress/`：站点配置
- `docs/artifacts/`：原始 artifacts 和总结
- `docs/memory/`：记忆快照
- `docs/projects/`：项目规划文档
