# 均值、方差、标准差

## 均值（Mean）

平均水平。所有值加起来除以个数。

```
数据：90, 60, 75, 88
均值 = (90 + 60 + 75 + 88) / 4 = 78.25
```

## 方差（Variance）

数据有多「散」——和均值差的平方的平均。

```
(90 - 78.25)² = 11.75² = 138
(60 - 78.25)² = (-18.25)² = 333
(75 - 78.25)² = (-3.25)² = 11
(88 - 78.25)² = 9.75² = 95

方差 = (138 + 333 + 11 + 95) / 4 = 144
```

平方是为了消掉正负号——偏离 +11.75 和 -18.25 都是「偏离」，平方后都变正。

> 方差的单位是「分²」，不直观。

## 标准差（Standard Deviation）

方差的平方根，单位回到原来的「分」。

```
标准差 = √144 = 12
```

意思：「这个班的成绩平均偏离均值 12 分左右」。

## 计算步骤

```
原始数据 ──→ 求平方差 ──→ 求平均 ──→ 开方 ──→ 标准差
90,60...     138,333...    144        12
```

## 在归一化中的作用

```
归一化后 = (原始值 - 均值) / 标准差

张三语文 = (90 - 78.25) / 12 = 0.98  ← 高于平均
李四语文 = (60 - 78.25) / 12 = -1.52 ← 低于平均
```

归一化后：正数 = 高于平均，负数 = 低于平均，绝对值大小 = 偏离多少个标准差。
